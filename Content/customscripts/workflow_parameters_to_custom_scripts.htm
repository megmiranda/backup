Passing Parameters to Custom Scripts used in TD Workflow

You can use environment variables to pass parameters and credentials to the custom script using _env. 

_env:
TD_API_KEY: ${secret:apikey}
TD_API_SERVER: ${secret:endpoint}
AWS_ACCESS_KEY_ID: ${secret:aws_access_key_id}
AWS_SECRET_ACCESS_KEY: ${secret:aws_secret_access_key}
