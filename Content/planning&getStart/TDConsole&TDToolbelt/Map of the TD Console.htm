Map of the TD Console

There are several interfaces found in the TD Console. Understanding where each interface is located and how to use them will allow you to understand more about what you can achieve with your Treasure Data CDP. The four main areas of the TD Console are seen below.

Global Navigation   In this menu, you can navigate to each of the following main sections of the TD Console: Integrations Hub, Data Workbench, Audience Studio and Jobs along with help and admin sections, all described in more detail below.

Local Navigation   This collapsible menu allows you to move within each of the main sections of the Global Navigation areas, for example Queries or Workflow within the Data Workbench.

Canvas   This is the area where you will find options for viewing, editing, querying and analyzing your TD Console sources and tasks.

Action Button   For each section, you will find action buttons in this area that allow you to start new actions such as creating queries or batch segments.

 

TD Console Global Navigation

The Global Navigation menu of the TD Console looks like this:

Integrations Hub is where you set up your data connections and import that data in bulk from various data sources.

Data Workbench is where you can query your data, view your databases, set up workflows for recurring queries, and set up your master segments.  

Audience Studio is a marketer-friendly interface where you can create a unified view of your customers using predefined master segments. You can create a new batch segment, view profiles, create predictive scoring models and set up new API Tokens.

Jobs allows you to view and monitor the status of all of your Treasure Data jobs. You can also view the status of these jobs under Sources.

Feedback is where you can share your ideas for enhancing our product.

Live Chat is where you can connect directly with our support team.

Get Help is where you are able to link to support documentation, learn keyboard shortcuts, submit support tickets, view product release notes and view the TD system status.

Control Panel is where you can add or delete users, create or edit permissions policies, view and edit the audit log, account info, login settings and the IP Whitelist as well as view utilization summaries.

My Settings is where you can edit your profile, create your API keys, and view which databases you own and have access to.
