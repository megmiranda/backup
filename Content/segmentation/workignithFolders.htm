Working with Segment Folders

Segment folders help you organize your work and create groupings that reflect your day-to-day processes.

  • Creating and Moving Folders
  • Renaming
  • Deleting

Creating and Moving Folders

  1. Navigate to Audience Studio > Segments.
  2. From Folder actions, select Create Folder.

  3. Type the name you would like to give to the folder.
  4. Create.
  5. From Folder Actions, select Move Segments to Folder.
  6. Select the source segment and all of its folders display.
  7. Select one of the segments to move.
  8. Select the destination folder where you would like to segment to be moved to.
  9. Select Move.

 

 

Renaming a Segment Folder

  1. Select a folder.
  2. Select the ...
  3. Select Rename Folder and type a name for the folder.

Deleting a Segment Folder

You can delete a segment folder only if it contains no segments.
