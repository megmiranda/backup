Working with Segments

With a segment selected, you can see a mini-dashboard of how many profiles have been added or removed from it over the course of time.

You can also access the following actions:

  • Editing Segment Rules
  • Editing Segment Name and Description
  • Cloning a Segment
  • Moving a Segment
  • Deleting a Segment
  • Creating Batch and Realtime Segments
  • Creating an Activation

Editing the Segment Name and Description

Opens an editor where you can mange the name and description of the segment. For example, use this option to rename the segment.

Cloning Segments

Selecting Clone Segment, opens an editor that allows you to copy the existing segment, rename it, and assign it a folder as necessary.

Moving a Segment

Selecting Move Segment to, opens an editor that allows you to assign the segment a new folder.

Deleting a Segment

Selecting Delete Segment, opens confirmation dialog. Confirm or cancel the deletion of the selected segment.

Creating Batch and Realtime Segments

These options take you to the editors where you can create new segments from scratch, by defining profile attributes, behaviors, and predictive scoring. For more information, search the support.treasuredata for Creating a New Segment.

Creating an Activation

Activating a segment exports its data to a target, for example Salesforce. For more information, search the support.treasuredata for Activating a new Segment.
